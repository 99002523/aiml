to run the auto script generator
3 parameters are mandatory
1. file name(python file itself).
2. path of the excel where test cases are defined 
3. link of the website on which automation has to performed
command is python autoscript_generator.py test_case.xlsx https://www.bigflix.com/
D:\aiml\lib\python>python autoscript_generator.py D:\aiml\excelsheet\test_case.xlsx https://www.bigflix.com/
